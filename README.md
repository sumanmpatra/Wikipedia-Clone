# Wikipedia Clone

This project is a simple clone of the Wikipedia website, designed to showcase basic web development skills. The project includes features such as article pages, search functionality, and a navigation menu, all styled to resemble the classic Wikipedia design.

## Features
- Homepage with a search bar to search for articles
- Static article pages mimicking Wikipedia's format
- Navigation menu for easy access to different categories

## Tech Stack
- **Frontend:**
  - HTML
  - CSS
